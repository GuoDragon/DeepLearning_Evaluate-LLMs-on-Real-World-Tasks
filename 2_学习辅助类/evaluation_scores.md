# 学习辅助类 LLM 输出评分报告

> 评分对象：`2_学习辅助类` 中 5 组 LLM 对 10 个深度学习学习辅助样例的基础 Prompt 与改进 Prompt 输出。  
> 评分来源：由现有初步报告 `分析報告.md` 第 7.2 节补回，用于补齐任务材料归档。  
> 评分原则：按事实正确性、完整性、清晰度、可靠性、知识覆盖度五项指标进行 1-5 分评价。

## 一、评分标准

| 指标 | 评价重点 |
|---|---|
| 正确性 | 概念解释是否事实正确，是否存在明显错误。 |
| 完整性 | 是否覆盖题目所需核心知识点和必要限定条件。 |
| 清晰度 | 表达是否面向本科生，结构是否清楚、易理解。 |
| 可靠性 | 是否避免幻觉、错误公式、过度自信或误导性表述。 |
| 知识覆盖度 | 是否体现相关深度学习知识的广度与联系。 |
