# Final Project：大语言模型能力评测

本仓库为深度学习课程 Final Project 提交材料，围绕多个大语言模型在不同任务场景下的表现进行对比评测。项目主要考察 LLM 在代码辅助、学习辅助与写作表达三类任务中的回答质量、Prompt 响应能力、稳定性和改进空间。

## 项目内容

本项目按照任务类型整理实验材料，每类任务均包含测试样例、基础 Prompt、改进 Prompt、模型输出记录与评分结果。最终报告在原始实验记录基础上，对不同模型的表现进行了定量评分和定性分析，并补充了典型案例、失败案例与项目反思。
